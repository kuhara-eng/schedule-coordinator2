# 日程調整ボード

PCのブラウザで `index.html` を開くと使えます。

## できること

- 9:00-18:00 の30分刻みで可否入力
- 複数の候補日を追加
- 開始日と終了日を指定して候補日を一括追加
- 複数の打ち合わせをタブで切り替え
- 可否欄をドラッグして連続入力
- Excelで文字化けしにくいCSVを書き出し
- 複数参加者の可否を1画面で集計
- 入力内容をブラウザ内に自動保存
- JSONでバックアップ/復元
- CSVで集計結果を書き出し

## 使い方

1. `index.html` をダブルクリックして開く
2. 打ち合わせ名を入力。複数ある場合は「打ち合わせを追加」でタブを増やす
3. 候補日を追加、または開始日と終了日を指定して期間をまとめて追加
4. 参加者を追加
5. 参加者ごとに「入力」を押して、時間枠をクリックまたはドラッグ
6. 「集計」タブで空いている人数を確認

JSONを書き出しておくと、別の日に読み込んで続きから編集できます。


## リアルタイム共有

Supabaseを使うと、同じ共有リンクを開いた人同士で入力内容が同期されます。

1. Supabaseで新しいプロジェクトを作る
2. `supabase-schema.sql` の内容を Supabase の SQL Editor で実行する
3. Supabase Dashboard の Database > Replication で `schedule_boards` のRealtimeを有効にする
4. `config.js` に Project URL と anon public key を入れる
5. `index.html` を開き、「共有リンクを作成」を押す
6. 「リンクをコピー」で参加者に共有する

共有リンクを知っている人は編集できます。公開範囲を厳密に制御したい場合は、ログイン制への変更が必要です。
